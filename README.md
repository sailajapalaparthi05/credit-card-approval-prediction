# 💳 Credit Card Approval Prediction System

## 📌 Project Overview

Credit Card Approval Prediction System is a Machine Learning based web application that predicts whether a user's credit card application will be **Approved** or **Rejected** based on applicant details.

The system uses applicant information and credit history data to train a machine learning model and provide quick approval predictions through an interactive web interface.

---

## 🎯 Objective

The main objective of this project is to build an intelligent prediction system that helps automate the credit card approval process using Machine Learning techniques.

---

## 🚀 Features

- User-friendly web interface
- Predicts Credit Card Approval status
- Machine Learning based prediction
- Displays prediction result with confidence score
- Fast and automated decision support
- Responsive design

## ⚙️ How It Works

1. User enters applicant details.
2. Input data is preprocessed.
3. Trained Machine Learning model analyzes the data.
4. System predicts:
   - ✅ Credit Card Approved
   - ❌ Credit Card Rejected

---

## 🤖 Machine Learning Model

Algorithm Used:

**Random Forest Classifier**

Random Forest is used because it provides good accuracy and handles both numerical and categorical data effectively.

---

## 📊 Dataset

The project uses credit card applicant data containing features such as:

- Gender
- Income details
- Education
- Employment status
- Family details
- Credit history
- Other applicant information
